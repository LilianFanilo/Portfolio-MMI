DataViz

*titre*

Le Projet DATAVIZ est un projet visant à réaliser un site web de visualisation graphique de données. Le but était de montrer un évolution de ces données. Ce fut un projet de groupe dont mon rôle a été de coder le graphique pour qu'il soit fonctionnel. il a fallu que je supperpose plusieurs tableaux de données différentes sur un même graphique et que je code une fonctionnalité permettant de zoomer.
